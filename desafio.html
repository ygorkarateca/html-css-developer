Hierarquia dentro do HTML

(Voltar)

A hierarquia em HTML é a organização dos elementos HTML em uma estrutura lógica, como uma árvore. Cada elemento tem seu lugar e função específica.

Isso permite que os navegadores interpretem corretamente o conteúdo e que os mecanismos de busca indexem a página de forma eficiente.

Em resumo: É a forma como os elementos HTML se relacionam entre si, criando uma estrutura clara e organizada para o conteúdo da página.

Listas Ordenadas e Não Ordenadas

(Voltar)

Listas em HTML servem para organizar informações de forma clara e concisa.

Listas ordenadas(OL) - numeram os itens em uma sequência específica, ideais para passos, rankings ou cronologias.

Listas não ordenadas(UL) - utilizam marcadores (bolinhas) para cada item, sem uma ordem específica, perfeitas para listas de compras ou itens relacionados.

Cada item da lista é definido pela TAG (LI)

O que são tags em HTML?

(Voltar)

Tags HTML são como as palavras-chave em um documento que instruem o navegador sobre como exibir o conteúdo de uma página web.

Elas são os blocos de construção básicos do HTML (HyperText Markup Language) e definem a estrutura, a formatação e o significado do conteúdo.

Imagine um livro: As tags são como as diferentes seções, capítulos e parágrafos que organizam o texto.

Elas indicam ao leitor (no caso, o navegador) que tipo de informação está sendo apresentada.
