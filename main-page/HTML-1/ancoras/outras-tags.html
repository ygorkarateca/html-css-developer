Utilizando outras tags

Principais tags utilizadas

  • (h1) - Define um título de nível 1 (o mais importante)
  • (p) - Indica um parágrafo de texto.
  • (img) - Insere uma imagem na página.
  • (a) - Cria um link para outra página.

Tags diversas

  • (abbr) - define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".
  • (font)* - tag foi usada no HTML 4 para especificar o rosto da fonte, tamanho da fonte e cor do texto
  • (del) - A tag define o texto que foi excluído de um documento. Os navegadores geralmente atingem uma linha através do texto excluído.

* Não suportado em HTML5 ou descontinuada
