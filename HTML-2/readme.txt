Aula 1 - Trabalhando com Formulários
Tag - <form>

<form name=""> - Nomear o seu formulário
<form action=""> - Direcionar o dados que contem no formulário
<form method=""> - Utiliza o metodo GET ou POST
<form autocomplete=""> - Utilizado para preencher os dados que já foram usado antes(ex. ON ou OFF)

Tipos de Target

<form target="_blank"> - A resposta é exibida em uma nova janela ou guia
<form target="_self"> - A resposta é exibida na janela atual

Tipos de Method

O que é method="GET" - Para buscas simples e quando a segurança não é crítica.
O que é method="POST" - Para formulários, envio de arquivos e quando a segurança é importante.

Tipo de ON

<form onsubmit=""> - Utilizado para adicionar um evento pos-envio do formulário

Tipos de input

<input> - Cria um campo para escrever lago
<input type=""> - Adiciona o tipo de input a ser utilizado(ex. text, number, password, checkbox)
<input type="text"> - Adiciona um campo para adicionar números e textos
<input type="number"> - Adiciona um campo para adicionar somente números
<input type="password"> - Adiciona um campo para senha, podendo adicionar qualquer tipo de caracteres
